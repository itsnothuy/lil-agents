## Key Observations

- **No `Package.swift`** — the project uses Xcode's native SPM integration, not a standalone package manifest.
- **No `Podfile`, `Cartfile`, or `Gemfile`** — Sparkle via SPM is the only external dependency.
- **No CI/CD configuration** — no GitHub Actions, no Fastlane, no build scripts.
- **No localization** — all strings are hardcoded in English.
- **No accessibility** — no VoiceOver labels, no dynamic type support.
